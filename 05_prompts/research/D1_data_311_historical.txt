Data Inventory D1 — RVA311 Historical Socrata Dataset (vgg4-hjn8)

Provide a detailed technical inventory of the RVA311 historical Socrata dataset. This dataset is the primary (and only publicly available) 311 data asset for Richmond.

Dataset details:
- Socrata dataset ID: vgg4-hjn8
- SODA API endpoint: data.richmondgov.com/resource/vgg4-hjn8.json
- Platform origin: SeeClickFix (January 2014 – August 2015 only)

Investigate and document:
1. Complete column/field schema: What are all the fields, their data types, and example values?
2. Date range: What is the exact first and last record date?
3. Total record count: How many requests are in the dataset?
4. Category distribution: What are the most common request types? How are categories structured?
5. Geographic fields: Are there latitude/longitude, address, neighborhood, or district fields?
6. Status fields: What request statuses are recorded (open, closed, resolved, etc.)?
7. Completeness: What percentage of records have complete data for key fields?
8. API limitations: Are there pagination constraints, rate limits, or query complexity limits?

Also address: Can the category taxonomy from 2014–2015 be used as a proxy for current 311 categories, or has the system changed significantly?

Output
- Complete field schema with types and example values
- Data quality assessment
- Practical guidance for a hackathon team using this dataset
- Limitations and what this dataset cannot tell you about current RVA311 operations
