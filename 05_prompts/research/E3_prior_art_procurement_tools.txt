Prior Art E3 — Contract Analytics and Procurement Transparency Tools

Investigate national prior art for government procurement transparency and contract analytics tools. Focus on tools built on public procurement data at the city or state level — analogous to what a hackathon team could build using the Richmond City Contracts Socrata dataset.

Key examples to investigate:
- USASpending.gov — federal contract transparency, open API
- OpenStates and similar state-level transparency tools
- City-level contract transparency dashboards built on Socrata or similar open data
- ProPublica Bailout Tracker, FEC data tools — examples of civic journalism-style data tools on public procurement/spending data
- Code for America procurement projects
- Civic hackathon projects that have built contract expiry trackers, procurement dashboards, or vendor debarment checkers
- Academic or NGO tools for analyzing government contracts (e.g., GovSpend, Periscope Data)
- PDF contract extraction tools — any open source or civic tools for extracting structured data from contract documents?

For each tool, document:
- Data sources used
- Technical approach
- Whether it is still active
- Key design decisions and lessons
- How it handles the "advisory only" vs. authoritative distinction

Output
- Inventory of prior art with technical approach and data sources
- Most applicable model for a Richmond contract expiry dashboard or cross-source finder
- Open source tools or libraries a hackathon team could leverage
